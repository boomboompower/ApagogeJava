# ApagogeJava

An implementation of Apagoge in Java


### Gradle usage
Add the Jitpack repository
```gradle
allprojects {
    repositories {
      ..
      maven { url 'https://jitpack.io' }
    }
}
```

Add the Apagoge dependancy
```gradle
dependencies {
    implementation 'com.github.boomboompower:ApagogeJava:master-SNAPSHOT'
}
```
